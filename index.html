PyAutoGUI Bot Template

A minimal baseline for GUI automation with PyAutoGUI

Reusable template for image-based desktop automation: locate/click helpers, configurable waits, random ranges, and file logging. Set PROJECT_ROOT once and add your images and flow.

  • Single place for project path (pyautogui_bot_config.py)
  • Image locate/click with confidence and human-like movement
  • Waits, random ranges, and timestamped log writing
